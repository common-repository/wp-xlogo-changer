Changes the logo on wp-login page.

== Description ==

View the plugin page for more information and discussion: [http://nexxoz.com/](http://nexxoz.com/)

WP xLogo Changer isa simple plugin to change the logo for wp-login page.

== Installation ==

Download and activate from the Plugins page of your WordPress site and then navigate to Settings -> WP xLogo Changer.

== Frequently Asked Questions ==

= Mixed Content over HTTPS, fix? =

This is a SSL problem. Go to Settings > General and change (Wordpress adress) & (Site adress) to https://your-site-name-here.com
